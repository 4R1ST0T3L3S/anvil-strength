# Abrir la web en local

He identificado que el proyecto utiliza **Vite** como herramienta de construcción. Para abrir la web localmente en Trae, seguiré estos pasos:

## Pasos de Ejecución

1.  **Instalación de dependencias**: Ejecutaré `npm install` para asegurar que todas las bibliotecas necesarias estén instaladas.
2.  **Iniciar servidor de desarrollo**: Ejecutaré el comando `npm run dev` para iniciar el servidor de Vite.
3.  **Abrir Vista Previa**: Una vez que el servidor esté listo (normalmente en `http://localhost:5173`), abriré la ventana de vista previa integrada en Trae para que puedas ver la aplicación.
