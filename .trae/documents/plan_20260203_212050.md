# Abrir Aplicación en Local

Para abrir la aplicación en tu entorno local, seguiré estos pasos:

1. **Verificar el servidor**: Comprobaré el estado del terminal donde ya se está ejecutando `npm run dev` para confirmar que el servidor de Vite está listo.
2. **Identificar la URL**: Obtendré la dirección local (normalmente `http://localhost:5173`).
3. **Abrir Vista Previa**: Utilizaré la herramienta de vista previa para abrir la aplicación directamente aquí en el editor.
