## Cómo quitar el fondo de una foto

1. Pasa la foto a Antigravity (el asistente IA) y pide: *"quita el fondo de esta foto"*
2. O usa https://www.remove.bg (gratis, muy preciso)
3. Guarda el resultado como PNG aquí con el nombre correcto

## Recomendaciones de foto
- Foto vertical (portrait), el atleta de cuerpo entero o hasta la cadera
- Fondo preferiblemente liso o de competición (para facilitar el recorte)
- Mínimo 800x1200px
